# 🎥 BLOOM (Video Recorder & Sharing App)
A modern video recording and sharing web application built with Next.js, Mux, and JavaScript.
Users can record videos directly in the browser, upload them, and share them instantly through a streaming-optimized platform.

# 🚀 Features

🎬 In-browser video recording

☁️ Video upload and processing

⚡ Fast video streaming

🔗 Shareable video links

📱 Responsive UI

🗂️ Video management

🔒 Secure upload flow

# 🧰 Tech Stack

Frontend: Next.js

Video Infrastructure: Mux

Language: JavaScript

API Routes: Next.js Server Functions

Storage & Streaming: Mux Video

# 📸 How It Works

User records a video using the browser camera.

The video is uploaded to Mux.

Mux processes and optimizes the video.

The app receives a playback URL.

Users can watch or share the video instantly.

# 💡 Inspiration

This project demonstrates how modern web frameworks like Next.js can integrate with video infrastructure platforms like Mux to build scalable video applications.
